To Run: python weatherApp.py

Description: 
Enter a city's name and the program will provide the forecast of the city by utilizing the OpenWeatherMap API to fetch and display current
weather information for the specified city. After obtaining user input and making an API request, the program extracts relevant weather
details from the JSON response, including description, temperature, feels-like temperature, and humidity, presenting a concise forecast summary.

Notes:
- JSON stands for javascript object notation
